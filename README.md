Arcs Hub is built using Glide and works seamlessly across devices. Users can browse services, submit listings, and explore categories with a clean, responsive interface.

### **1. Business owners add their services**
Service providers can submit:
- Service name  
- Category  
- Description  
- Contact information  
- Optional images

Users can:
- Choose a service category  
- View all services within that category  
- Open detailed service listings  
- Access contact details directly

- Automatic mobile + desktop responsiveness
